Radio buttons group

Wrap a series of buttons with .btn in .radio-buttons-group. Add on optional JavaScript radio and checkbox style behavior with our buttons plugin.

Each radio button group behave just like native input in JS so you can subscribe for its change event which will be fired with the value of data-value attribute of selected button:

$('.radio-buttons-group').on('change', function(e, value) {
    console.log('selected value is ' + value);
});
