Empathy Building Game for Accessibility

Consider for a moment, that you are blind.

You heard your colleagues mention a local arts and music festival to you in the break room. You barely caught the name of it as you rushed off to the next meeting, and you think you have the correct website on the screen in front of you now.

You love people, and you love music; It would be great if you could take your guide dog; You want to get a feel of what kind of event this is so you can plan ahead.

Blind people don't use computers the same way sighted people do; Often, they use the keyboard to navigate as a sighted user would do with a mouse. The keyboard sends messages to the installed screen reader application, which reads out information about the screen they are looking at to the person, piece by piece.

Keeping this in mind, try navigating this website using only the keyboard to find the information you need.

Press Tab to start navigating. What screen readers see will show as text here.

Did you find yourself doing any of the following?

  • Memorising the steps when you weren't able to get to the button you wanted to with the keyboard ... not so successfully
  • Needing to go back constantly to remind yourself what the screen reader "said".

The digital landscape can be a hostile, lonely place when you're not a sighted person. Accessible applications can be an ally by ensuring that screen readers can talk to it.

You can make a difference today by:

  • Go to your favourite website and do an accessibility check.

    • Get axe-core browser extension by deque
    • Get accessibilityinsights.io by Microsoft
    • Is it not up to scratch? Send their administrator a friendly message.

  • Become an advocate - educate yourself and your colleagues so you can champion the change.

    • deque university offers individual and corporate packages
    • WCAG documentation defines the global standard for accessible applications.
